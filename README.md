# TravelBloom Recommendations

TravelBloom is a simple travel recommendation website for the Coursera final project. It includes a home page, about page, contact page, navigation bar, search feature, and recommendation results with images.

## Features

- Home page with introduction and call-to-action button
- About Us page with company information and team members
- Contact Us page with an email form
- Navigation bar with Home, About Us, and Contact Us links
- Search and Clear buttons
- Beach recommendations with two images
- Temple recommendations with two images
- Country recommendations for Japan, Australia, and Brazil with two images each
